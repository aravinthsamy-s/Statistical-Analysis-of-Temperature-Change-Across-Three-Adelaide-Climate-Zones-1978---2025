# Adelaide Temperature Analysis (1978–2025)

This project analyses long-term temperature trends across three Adelaide climate zones:

* Adelaide Airport
* Edinburgh RAAF
* Mount Barker

The analysis uses statistical techniques including:

* ANOVA
* Mann–Kendall trend test
* Sen’s slope estimation
* Linear regression
* Forecast modelling

## Files Included

* `temperature_analysis code.Rmd` — R Markdown analysis code

## Software Used

* R
* ggplot2
* dplyr
* Kendall
* trend
* broom

## Purpose

The project investigates warming trends across Adelaide regions between 1978 and 2025 and provides future temperature forecasts.
